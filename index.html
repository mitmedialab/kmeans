K-means Custering

K-means is a kind of clustering model that forms a fixed number (K) of collections or groups of data points based on feature similarities from a dataset of unlabelled data points. The number of clusters identified from data by algorithm is represented by 'K' in K-means.

Interactive tool

Let's try plotting a few points in the canvas on the right and cluster them.

Instructions:

1. Click on the canvas to add data points.
2. You can use the pencil icon to draw to guess the clusters.
3. Click the button Cluster, to cluster them.
4. Adjust the number of clusters with the slider.
